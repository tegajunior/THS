
## Description

A simple appointment booking system built with Next.js

View live demo at:
https://ths-mu.vercel.app/

## Installation

```bash
$ npm install
```

## Setting up the environment

```bash
# Create a .env file in the root of the project with these variables for the server

$ DB_URL =
$ DB_HOST =
$ DB_USER =
$ DB_PASSWORD =
$ DB_DATABASE =


# Create a .env.local file in the root of the project with these variables

$ DB_URL =
$ DB_HOST =
$ DB_USER =
$ DB_PASSWORD =
$ DB_DATABASE =

```

## Creating migration

```bash
# Migrate down
$ npm run unmigrate

# Migrate up
$ npm run migrate

# Seed
$ npm run seed

```

## Running the app

```bash
# Development
$ npm run dev

```

## Running tests

```bash
# Test
$ npm run test

```
## Running docker container

```bash
# Docker up
$ docker-compose up

```
